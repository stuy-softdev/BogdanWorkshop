Get VS Post

GET: Used for reading/retrieving data. In our code, requests.arg does this by getting it from the URL. Thats why it can return requests.arg['username']

Post: Used for sending data that can change something, like forms or uploads. Instead of requests.args, its request.form or other. POST dont show up in URL

In our code, both '/' and /auth are only using GET requests. It pulls the string from the URL.

To switch to post, we just need to change themethods=['GET', 'POST'] as shown in the previous code. Then instead of requests.args, we would use request.form to get the submitted fields
